Medidas Absolutas e Relativas

Aprendendo Margin, Padding e Unidades de Medida em CSS

1. Medidas Absolutas (PX - Pixels)

Medidas absolutas têm tamanho fixo e não mudam com o redimensionamento da tela. A unidade mais comum é o pixel (px).

Dica: Use medidas absolutas quando precisa de precisão e tamanhos fixos que não devem variar.

Exemplos com Pixels:

100px × 100px
200px × 120px
300px × 150px
.box-pequeno {
    width: 100px;
    height: 100px;
}

2. Medidas Relativas (%, EM, REM)

Medidas relativas se adaptam ao tamanho do container ou da tela. São mais flexíveis para design responsivo.

A) Percentual (%)

O percentual é relativo ao tamanho do elemento pai.

25% da largura
50% da largura
75% da largura
100% da largura
.box-25 {
    width: 25%;
    height: 80px;
}
💡 Dica: Use percentuais para layouts fluidos que se adaptam ao tamanho da tela.

B) Unidades EM e REM

EM: Relativo ao tamanho da fonte do elemento pai.

REM: Relativo ao tamanho da fonte do elemento raiz (html).

1.5em de padding
1.5rem de padding
Unidade Descrição Quando Usar
px Pixels - Absoluto Bordas, sombras, elementos que não mudam
% Percentual - Relativo ao pai Layouts responsivos, larguras
em Relativo à fonte do pai Espaçamentos dentro de componentes
rem Relativo à fonte raiz (html) Design escalável e consistente

3. Margin (Espaço Externo)

Margin é o espaço fora do elemento. É a distância entre o elemento e seus vizinhos.

Demonstração de Diferentes Margins:

margin: 10px
margin: 20px
margin: 40px
.box {
    margin: 20px;           /* Todos os lados */
    margin: 10px 20px;      /* Vertical | Horizontal */
    margin: 10px 20px 30px 40px;  /* Top | Right | Bottom | Left */
}
⚠️ Atenção: Margens de elementos adjacentes podem se "colapsar" (fundir), resultando em uma única margem maior.

4. Padding (Espaço Interno)

Padding é o espaço dentro do elemento, entre o conteúdo e a borda.

Demonstração de Diferentes Paddings:

padding: 5px
padding: 20px
padding: 40px
.box {
    padding: 20px;           /* Todos os lados */
    padding: 10px 20px;      /* Vertical | Horizontal */
    padding: 10px 20px 30px 40px;  /* Top | Right | Bottom | Left */
}
💡 Dica: Use padding para criar espaço interno e melhorar a legibilidade do conteúdo.

5. Comparação: Margin vs Padding

Aspecto Margin Padding
Localização Espaço EXTERNO (fora do elemento) Espaço INTERNO (dentro do elemento)
Afeta background? Não (transparente) Sim (coberto pelo background)
Colapsa? Sim (entre elementos) Não
Quando usar Distância entre elementos Espaço do conteúdo da caixa

Exemplo Combinado:

Margin + Padding
(margin: 15px, padding: 25px)

7. Box Model (Modelo de Caixa)

Todo elemento HTML é uma caixa, composta por:

Content (Conteúdo)
↓ Padding (Espaço interno) ↓
↓ Border (Borda) ↓
↓ Margin (Espaço externo) ↓ 
/* Box Model */
Content → Padding → Border → Margin

Exemplo:
width: 200px           (largura do conteúdo)
padding: 20px          (espaço interno)
border: 2px solid      (borda)
margin: 15px           (espaço externo)

Largura TOTAL = 200 + (20×2) + (2×2) + (15×2) = 294px
💡 Dica: Use box-sizing: border-box; para facilitar cálculos de dimensão. Assim, padding e border não aumentam o tamanho final da caixa.

📝 Resumo Rápido

Conceito Exemplo Resultado
Medida Absoluta width: 200px; Sempre 200 pixels, sem importar o contexto
Medida Relativa width: 50%; Metade do tamanho do elemento pai
Margin margin: 20px; Espaço externo de 20px em todos os lados
Padding padding: 15px; Espaço interno de 15px em todos os lados
